# todoom
スコアで競い合う新感覚TODOアプリ。

### クローン

```bash
$ git clone https://github.com/tyokinuhata/todoom.git
```

### 環境構築

```bash
$ cd todoom
$ make setup
```

### 起動方法

```bash
$ make vup
$ npm run watch
$ open http://{your_private_ip_address}:8000  // or open localhost:8000
```

### コードの圧縮(フロントエンド)

```bash
$ npm run production
```
